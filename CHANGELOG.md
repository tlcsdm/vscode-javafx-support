# Change Log

## [1.0.3] - 2026-04-01

### Added
- Enhanced FXML syntax highlighting with improved grammar support
- Added `<?language ?>` processing instruction support
- Distinguished property element tags from class element tags for better visual clarity
- Improved import statement highlighting with package path and wildcard distinction
- Added support for both single-line and multi-line XML comment highlighting

## [1.0.1] - 2026-03-11

### Added
- Support for displaying FXML files in the Outline view

## [1.0.0] - 2026-02-26

### Added
- Initial release
- FXML syntax highlighting with TextMate grammar
- FXML language mode registration
- Scene Builder integration with configurable path
- "Open in Scene Builder" context menu for FXML files
- FXML document formatter
- FXML → Controller navigation (fx:controller, fx:id, onAction)
- Controller → FXML navigation (@FXML annotated fields and methods)
- Internationalization support (English, Chinese, Japanese)
